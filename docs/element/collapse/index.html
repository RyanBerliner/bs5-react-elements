Collapse

Wrapper for the Bootstrap collapse component.

Example Usage

import React from "react";
import {Collapse} from "bs5-react-elements";

function ExampleUsage() {
  return <>
    <button
      data-bs-toggle="collapse"
      data-bs-target="#example-usage"
      aria-expanded="false"
      aria-controls="example-usage"
    >
      Toggle
    </button>
    <Collapse id="example-usage" className="collapse">
      Lorem ipsum dolor.
    </Collapse>
  </>;
}

Important! Bootstrap 5.0.x users, if you wish to configure your collapse, you must use the config prop, as data attributes are not supported in this version. This bug is fixed in later versions of boostrap (5.1.x, 5.2.x, etc).
